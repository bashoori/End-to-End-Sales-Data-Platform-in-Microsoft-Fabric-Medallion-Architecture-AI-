# End-to-End-Sales-Data-Platform-in-Microsoft-Fabric-Medallion-Architecture-AI

## Problem

In many organizations, sales data comes from multiple sources and lacks structure, reliability, and real-time visibility.  

This leads to:
- inconsistent reporting  
- delayed decision-making  
- lack of trust in data  

The goal of this project is to design a modern data platform using Microsoft Fabric that ingests, transforms, and delivers reliable sales insights.
